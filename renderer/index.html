Terms of Use

Scroll to the bottom to enable

FLUX is a desktop media manager that bundles best-of-breed open-source tools (yt-dlp, ffmpeg, Chromaprint, WaveSurfer) behind a single coherent UI. It lets you fetch media from public sources, organise it locally, and convert or edit it — without juggling command lines or third-party websites.

Fetch

Paste any media URL (YouTube, podcast feeds, RSS, internet radio, public Spotify tracks for metadata-only lookup) and FLUX resolves the best playable stream via yt-dlp. RSS feeds support auto-update; podcasts can be played in-app or downloaded as MP3/MP4.

Load

Build playlists from local files or streaming URLs and export them as M3U for phones or smartwatches. Queue dozens of downloads at once and run them sequentially, or import an existing CSV / TXT list from the Queue tab.

Use

The Tag Editor reads ID3v2 / FLAC / M4A metadata, looks up missing fields on MusicBrainz, fetches cover art automatically, and writes the result back. Auto-tag fires on every successful audio download — finished files arrive with title, artist, album and artwork already populated.

Xtract

Trim a section of any audio or video file with a draggable waveform region, apply optional fade-in / fade-out, and convert to another container — all in one click. ffmpeg does the heavy lifting; lossless stream-copy when the format does not change, re-encode only when needed.

USE IT LEGALLY

FLUX never bypasses DRM and never circumvents access controls. You are responsible for the content you choose to download, store, or distribute. Use it for your own legitimate library: personal recordings, public-domain works, content you have a licence for, or material whose owners have explicitly authorised redistribution.

Support us

FLUX is free and open-source. If it earns a place in your workflow, consider supporting its development — donations cover hosting, signing certificates, and the time spent maintaining the project.

              Bulk rename

              Pattern tokens: {name} {ext} {n} {nn} {nnn} {date} {time} {w} {h}. Example: vacation-{nn}.{ext}

              Bulk convert

              Convert to another format. JPG/WebP/AVIF accept a quality (1-100). PNG is lossless.

              Bulk resize

              Max width / max height (in px) or scale percentage. Aspect ratio preserved. Leave a field at 0 to ignore.

              EXIF

              Strip all metadata (privacy: removes GPS, camera info, date, etc.) or auto-rotate to the correct orientation per the existing EXIF Orientation tag.

              HEIC → JPG

              Convert iPhone .heic/.heif files to JPG. The originals are kept; converted copies are placed next to them.

              Apply text watermark

              Overlays bold text on every selected image. Configure font size, colour, opacity, position (9-point grid), and optional drop shadow.

              Position

              Compress to target size

              Binary-search the encoder quality until the output file fits the target size. Useful for "all photos under 500 KB for the blog" workflows.

              Find duplicates

              Perceptual hash (dHash) comparison — catches the same image in different resolutions, formats, or with minor edits. Threshold is Hamming distance (0-64 bits); 8 is a sensible default.

              Similar photos · pick the best

              Groups visually-similar shots (bursts, near-duplicates) into a gallery and suggests the sharpest, highest-resolution one to keep. Sensitivity is the perceptual distance (0-64): higher groups looser sets, lower is stricter. Review, choose a keeper per group, then trash the rest.

              Organize into folders

              Move photos into dated subfolders by capture date (EXIF, with file date as fallback). Tokens: {year} {month} {day} {date} {make} {model} {camera} {ext}. Example: {year}/{month}. Files are moved.

              Timelapse / slideshow

              Combine the loaded images into a video. Each image becomes one frame; FPS controls how fast the sequence plays back. Order = list order. All images should ideally share the same resolution.

              Combine to PDF

              Bundle the loaded images into a single PDF, one image per page. Page size is A4 and each image is centred at its natural aspect — useful for delivering screenshot decks or photo contact sheets.

              Output

                Network proxy (SOCKS5)

                Routes all FLUX traffic (IRC, yt-dlp downloads, RSS, MusicBrainz, Spotify, etc.) through the proxy. Examples: 127.0.0.1:9050 for Tor, or a SOCKS5 endpoint from your VPN provider.

                Music Recognition

                Pick the backend used by the song-identify button. Shazamio uses Shazam's public API (no key, broad catalogue). AcoustID uses Chromaprint + MusicBrainz (free key, more privacy).

                Register an Application (not the personal API key page) to get an 8-char client key.

                Subtitles (OpenSubtitles)

                FLUX can find subtitles online for any local video via OpenSubtitles.com. Free registration required — get an API key from opensubtitles.com/consumers (free tier: 200 downloads/day). Without a key, the "Find online" button stays disabled.
